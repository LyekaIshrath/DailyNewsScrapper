# 📰 News Web Scraper with Email Automation

## 📌 Project Overview

This project is a Python-based automation application that scrapes the latest news headlines from a news website and automatically sends them to a specified email address.

The project demonstrates web scraping, email automation, and Python programming concepts, making it suitable for beginners and a great addition to a software development or automation portfolio.

---

## 🚀 Features

- Scrapes latest news headlines from a website
- Extracts headline titles
- Sends news headlines automatically via email
- Simple and easy-to-understand Python code
- Secure credential management using environment variables
- Beginner-friendly project structure

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/news-web-scraper-email.git
```

### 2. Navigate to the project folder

```bash
cd news-web-scraper-email
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

---

## 🔐 Configure Environment Variables

Create a `.env` file in the project directory.

Example:

```
EMAIL_ADDRESS=your_email@gmail.com
EMAIL_PASSWORD=your_app_password
RECEIVER_EMAIL=receiver@gmail.com
```

> Never upload your `.env` file to GitHub.

---

## ▶️ Run the Project

```bash
python main.py
```

If everything is configured correctly, the latest news headlines will be sent to your email.
